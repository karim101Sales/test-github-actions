\begin{tcolorbox}[enhanced jigsaw, breakable, colback=white, coltitle=black, arc=.35mm, bottomtitle=1mm, rightrule=.15mm, colbacktitle=quarto-callout-note-color!10!white, colframe=quarto-callout-note-color-frame, toptitle=1mm, leftrule=.75mm, bottomrule=.15mm, opacitybacktitle=0.6, toprule=.15mm, title=\textcolor{quarto-callout-note-color}{\faInfo}\hspace{0.5em}{Quarto 1.3 Feature}, titlerule=0mm, left=2mm, opacityback=0]

This post is part of a series highlighting new features in the 1.3
release of Quarto. Get the latest release at
\url{https://quarto.org/docs/download}.

\end{tcolorbox}

Starting in Quarto 1.3, HTML pages (either standalone or in a website)
can automatically include links to other formats specified in the
document front matter. For example, the following document front matter:

\begin{Shaded}
\begin{Highlighting}[]
\FunctionTok{title}\KeywordTok{:}\AttributeTok{ Sample Page}
\FunctionTok{author}\KeywordTok{:}\AttributeTok{ Norah Jones}
\FunctionTok{date}\KeywordTok{:}\AttributeTok{ last{-}modified}
\FunctionTok{toc}\KeywordTok{:}\AttributeTok{ }\CharTok{true}
\FunctionTok{format}\KeywordTok{:}\AttributeTok{ }
\AttributeTok{  }\FunctionTok{html}\KeywordTok{:}\AttributeTok{ default}
\AttributeTok{  }\FunctionTok{ipynb}\KeywordTok{:}\AttributeTok{ default}
\end{Highlighting}
\end{Shaded}

Results in an HTML page that includes a link to the additional notebook
format in the right margin below the table of contents:

\includegraphics{other-format.png}

If a table of contents is enabled for the page, the additional formats
will be automatically placed within the table of contents as a new
section. If no table of contents is displayed, the additional formats
will be displayed in the right margin at the top of the document.

Links to additional formats are displayed by default, but you can
control whether they are shown or even which specific formats are
included with the \texttt{format-links} YAML option.

Read more about this feature on the
\href{../../../../docs/prerelease/1.3/multi-format.html}{Multi-format
page} of the pre-release highlights.
